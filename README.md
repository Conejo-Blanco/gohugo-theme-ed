# Ed
A theme for [Hugo](http://gohugo.io), a framework for building websites.

## Configuration

### `config.toml` example

```toml
baseURL = 'http://example.org/'
languageCode = 'en-us'
title = 'My New Hugo Site'
theme = "ed"

[Params]
    # Color scheme. Options: red, orange, magenta, cyan, blue, brown
    color_scheme = ''
```
